Materi 1: Menjelajahi Editor UI

Di materi ini, Anda akan belajar cara menavigasi Editor UI (User Interface) n8n. Kita akan menjelajahi kanvas dan menunjukkan arti setiap ikon serta di mana menemukan hal-hal yang Anda perlukan saat membangun workflow di n8n.

Versi n8n
Materi ini didasarkan pada n8n versi 1.82.1. Pada versi lain, beberapa antarmuka pengguna mungkin terlihat berbeda, tetapi ini seharusnya tidak memengaruhi fungsionalitas inti.

Memulai

Mulailah dengan menyiapkan n8n.

Kami merekomendasikan memulai dengan n8n Cloud, solusi yang di-host yang tidak memerlukan instalasi dan menyertakan uji coba gratis.

Pengaturan Alternatif
Jika n8n Cloud bukan pilihan yang baik untuk Anda, Anda dapat melakukan self-host dengan Docker. Ini adalah opsi lanjutan yang direkomendasikan hanya untuk pengguna teknis yang terbiasa dengan layanan hosting, Docker, dan baris perintah (command line).

Untuk detail lebih lanjut tentang berbagai cara menyiapkan n8n, lihat dokumentasi platform kami.

Setelah n8n berjalan, buka Editor UI di jendela browser. Masuk ke instans n8n Anda. Pilih Overview lalu Create Workflow untuk melihat kanvas utama.

Tampilannya akan terlihat seperti ini:

Gambaran Editor UI n8n

Pengaturan Editor UI

Editor UI adalah antarmuka web tempat Anda membangun workflow. Anda dapat mengakses semua workflow dan kredensial Anda, serta halaman dukungan, dari Editor UI.

Panel Sisi Kiri

Di sisi kiri Editor UI, terdapat panel yang berisi fungsionalitas inti dan pengaturan untuk mengelola workflow Anda. Buka dan tutup panel ini dengan memilih ikon panah kecil.

Menu Sisi Kiri Editor UI

Panel ini berisi bagian-bagian berikut:

  • Overview: Berisi semua workflow, kredensial, dan eksekusi yang dapat Anda akses. Selama kursus ini, buat workflow baru di sini.
  • Personal: Setiap pengguna mendapatkan proyek personal default. Jika Anda tidak membuat proyek kustom, workflow dan kredensial Anda disimpan di sini.
  • Projects: Memungkinkan Anda mengelompokkan workflow dan kredensial. Anda dapat menetapkan peran (role) ke pengguna dalam proyek untuk mengontrol apa yang dapat mereka lakukan. (Tidak tersedia di edisi Community).
  • Admin Panel: (Hanya n8n Cloud) Akses penggunaan instans n8n, tagihan, dan pengaturan versi Anda.
  • Templates: Kumpulan workflow siap pakai. Tempat yang bagus untuk memulai dengan kasus penggunaan umum.
  • Variables: Digunakan untuk menyimpan dan mengakses data tetap di seluruh workflow Anda. (Tersedia di Paket Pro dan Enterprise).
  • Insights: Menyediakan analitik dan wawasan tentang workflow Anda.
  • Help: Berisi sumber daya seputar produk dan komunitas n8n.
  • Whatâ€™s New: Menampilkan pembaruan dan fitur produk terbaru.

Bilah Atas (Top Bar)

Bilah atas Editor UI berisi informasi berikut:

Bilah Atas Editor UI

  • Nama Workflow: Secara default, n8n menamai workflow baru sebagai "My workflow", tetapi Anda dapat mengedit namanya kapan saja.
  • + Add Tag: Tag membantu Anda mengatur workflow berdasarkan kategori, kasus penggunaan, atau apa pun yang relevan bagi Anda. Tag bersifat opsional.
  • Tombol Inactive/active: Tombol ini mengaktifkan atau menonaktifkan workflow saat ini. Secara default, workflow dinonaktifkan.
  • Share: Anda dapat berbagi dan berkolaborasi dengan orang lain pada workflow (di paket Starter, Pro, dan Enterprise).
  • Save: Tombol ini menyimpan workflow saat ini.
  • History: Setelah Anda menyimpan workflow, Anda dapat melihat versi sebelumnya di sini.

Kanvas (Canvas)

Kanvas adalah latar belakang kotak-kotak abu-abu di Editor UI. Ini menampilkan beberapa ikon dan node dengan fungsionalitas berbeda:

Kanvas Workflow

  • Tombol untuk memperbesar (zoom) kanvas agar pas di layar, memperbesar atau memperkecil kanvas, mengatur ulang zoom, dan merapikan node di layar.
  • Tombol untuk Execute workflow setelah Anda menambahkan node pertama Anda. Saat Anda mengkliknya, n8n mengeksekusi semua node di kanvas secara berurutan.
  • Tombol dengan tanda + di dalamnya. Tombol ini membuka panel node.
  • Tombol dengan ikon catatan (note). Tombol ini menambahkan catatan tempel (sticky note) ke kanvas (terlihat saat mengarahkan kursor ke ikon + di kanan atas).
  • Tombol berlabel Ask Assistant muncul di sisi kanan kanvas. Anda dapat meminta bantuan AI Assistant untuk membangun workflow.
  • Kotak putus-putus dengan teks "Add first step." Di sinilah Anda menambahkan node pertama Anda.
Memindahkan Kanvas
Anda dapat memindahkan kanvas workflow dengan tiga cara:
  • Pilih Ctrl+Tombol Kiri mouse di kanvas dan gerakkan.
  • Pilih Tombol Tengah mouse di kanvas dan gerakkan.
  • Letakkan dua jari di touchpad Anda dan geser.

Jangan khawatir tentang eksekusi dan aktivasi workflow untuk saat ini; kami akan menjelaskan konsep-konsep ini nanti dalam kursus.

Node

Anda dapat menganggap node sebagai blok bangunan yang melayani fungsi berbeda yang, ketika disatukan, membentuk mesin yang berfungsi: workflow otomatis.

Sebuah Node

Node adalah langkah individual dalam workflow Anda: yang (a) memuat, (b) memproses, atau (c) mengirim data.

Berdasarkan fungsinya, n8n mengklasifikasikan node ke dalam empat jenis:

  • Node Aplikasi atau Aksi: Menambah, menghapus, dan mengedit data; meminta dan mengirim data eksternal; dan memicu peristiwa di sistem lain.
  • Node Pemicu (Trigger): Memulai workflow dan memasok data awal.
  • Node Inti (Core): Bisa berupa node pemicu atau aplikasi. Jika sebagian besar node terhubung ke layanan eksternal tertentu, node inti menyediakan fungsionalitas seperti logika, penjadwalan, atau panggilan API generik.
  • Node Kluster: Adalah grup node yang bekerja sama untuk menyediakan fungsionalitas dalam workflow, terutama untuk workflow AI.

Menemukan Node

Anda dapat menemukan semua node yang tersedia di panel node di sisi kanan Editor UI. Ada tiga cara untuk membuka panel node:

  • Klik ikon + di sudut kanan atas kanvas.
  • Klik ikon + di sisi kanan node yang ada di kanvas (node yang ingin Anda tambahi node lain).
  • Tekan tombol Tab di keyboard Anda.

Panel Node

Di panel node, perhatikan bahwa saat menambahkan node pertama, Anda akan melihat berbagai kategori node pemicu. Setelah Anda menambahkan node pemicu, Anda akan melihat panel node berubah untuk menampilkan node Advanced AI, Actions in an App, Data transformation, Flow, Core, dan Human in the loop.

Jika Anda ingin menemukan node tertentu, gunakan input pencarian di bagian atas panel node.

Menambahkan Node

Ada dua cara untuk menambahkan node ke kanvas Anda:

  1. Pilih node yang Anda inginkan di panel node. Node baru akan otomatis terhubung ke node yang dipilih di kanvas.
  2. Seret (drag) dan lepas (drop) node dari panel node ke kanvas.

Tombol Node

Jika Anda mengarahkan kursor ke sebuah node, Anda akan melihat tiga ikon muncul di atasnya:

  • Jalankan node (Ikon Play)
  • Nonaktifkan/Aktifkan node (Ikon Power)
  • Hapus node (Ikon Tempat Sampah)

Juga akan ada ikon elipsis (tiga titik), yang membuka menu konteks berisi opsi node lainnya.

Ringkasan

Di materi ini, Anda telah belajar cara menavigasi Editor UI, arti ikon-ikonnya, cara mengakses panel sisi kiri dan panel node, serta cara menambahkan node ke kanvas.
