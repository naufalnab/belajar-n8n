Materi 16: Memproses Berbagai Tipe Data

Di bab ini, Kita akan belajar cara menangani tipe data yang lebih spesifik: HTML, XML, Tanggal & Waktu, dan Data Biner (file). Ini adalah kemampuan penting untuk menghubungkan sistem yang berbeda.

Bagian 1: HTML dan XML

Seringkali kita perlu mengambil data dari website (HTML) atau layanan lama yang menggunakan XML. n8n memiliki node khusus untuk ini.

Latihan 1: Web Scraping Lanjutan (HTML)

Mari kita coba mengambil data dari blog n8n. Tidak hanya satu, tapi semua artikel di halaman depan, dan kita ingin mengambil Judul beserta Link-nya.

Tantangan:

  1. Gunakan HTTP Request untuk mengambil https://blog.n8n.io/.
  2. Gunakan node HTML untuk mengekstrak data.
  3. Pastikan Kita mendapatkan Judul DAN Link URL untuk setiap artikel.
Tips:
  • Gunakan opsi Return Array untuk mengambil banyak item.
  • Kita bisa menambahkan lebih dari satu "Value" di bagian Extraction Values.
  • Untuk mengambil Link, ubah Return Value menjadi Attribute dan nama atributnya adalah href.

Bagian 2: Tanggal, Waktu, dan Penundaan

Bekerja dengan tanggal seringkali membingungkan karena banyaknya format (DD-MM-YYYY, ISO, Unix Timestamp, dll).

n8n menyediakan node Date & Time untuk mempermudah konversi dan kalkulasi tanggal. Selain itu, ada node Wait untuk menunda eksekusi (misal: menunggu API lain selesai memproses).

Latihan 3: Manipulasi Tanggal & Wait Node

Skenario latihan yang cukup kompleks:

  1. Ambil data pelanggan dari node Customer Datastore.
  2. Bulatkan tanggal pembuatan (created date) ke Akhir Bulan.
  3. JIKA tanggal baru tersebut setelah tahun 1960:
    • Tunggu selama 1 menit.
    • Set field baru outputValue.

Full Flow

Membulatkan tanggal ke akhir bulan.

3. IF Node:

  • Resume: After Time Interval

Bagian 3: Data Biner (File)

Sejauh ini kita hanya bekerja dengan teks (JSON). Tapi bagaimana dengan gambar, PDF, atau file Excel? Di n8n, ini disebut Binary Data.

Node-node penting untuk file:

  • HTTP Request: Bisa mendownload file dari internet.
  • Read/Write Files from Disk: Membaca/menulis file di server lokal (Hanya Self-Hosted).
  • Extract From File: Membaca isi file (misal teks dari PDF).

Latihan 4: PDF ke JSON

Tantangan:

  1. Download file PDF dari URL: https://media.kaspersky.com/pdf/Kaspersky_Lab_Whitepaper_Anti_blocker.pdf
  2. Ekstrak teks dari PDF tersebut menjadi JSON.
