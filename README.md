# Belajar n8n

Selamat datang di repositori catatan pembelajaran n8n saya!

Repositori ini berisi kumpulan catatan, cuplikan kode, dan dokumentasi alur kerja (workflow) yang saya buat selama proses mempelajari n8n, mulai dari dasar hingga topik-topik lanjutan.

## Tujuan

Tujuan dari repositori ini adalah:
* Mendokumentasikan pemahaman saya tentang konsep-konsep n8n.
* Menyimpan contoh workflow yang dapat digunakan kembali.
* Mencatat solusi untuk masalah-masalah umum yang dihadapi.
