Materi 4: Merancang Alur Kerja (Workflow)

Setelah kita tahu apa yang ingin diotomatiskan oleh Nathan (dari Materi 3), saatnya kita merancang solusi langkah demi langkah. Inilah proses di mana kita menerjemahkan "masalah" menjadi "alur kerja n8n".

Membedah Rencana Nathan

Berdasarkan percakapan kita, kita tahu bahwa Nathan perlu mengambil tindakan berikut untuk mencapai tujuannya:

  1. Mengambil data yang relevan (ID pesanan, status, nilai, nama karyawan) dari data warehouse.
  2. Memfilter pesanan berdasarkan statusnya (Processing atau Booked).
  3. Menghitung total nilai dari semua pesanan yang statusnya Booked.
  4. Memberi tahu anggota tim tentang pesanan Booked di channel Discord perusahaan.
  5. Memasukkan detail pesanan Processing ke Airtable untuk ditindaklanjuti.
  6. Menjadwalkan alur kerja ini agar berjalan setiap Senin pagi.

Merancang Alur di n8n

Alur kerja Nathan melibatkan pengiriman data dari data warehouse internal perusahaan ke dua layanan eksternal:

  • Discord
  • Airtable

Sebelum itu, data harus "diolah" terlebih dahulu menggunakan fungsi umum (seperti pemfilteran kondisional, kalkulasi, dan penjadwalan).

Berita baiknya, n8n menyediakan integrasi untuk semualangkah ini. Jadi, alur kerja Nathan di n8n akan terlihat seperti ini secara visual:

Rencana alur kerja kita. Satu input (Trigger), bercabang menjadi dua output (Discord & Airtable).

Rencana Pembangunan

Kita akan membangun alur kerja lengkap ini dalam 8 langkah ke depan (di materi-materi selanjutnya):

  1. Materi 5: Mengambil data dari data warehouse.
  2. Materi 6: Memasukkan data ke Airtable.
  3. Materi 7: Memfilter pesanan.
  4. Materi 8: Mengatur nilai untuk pesanan "Processing".
  5. Materi 9: Menghitung pesanan "Booked".
  6. Materi 10: Memberi notifikasi ke tim.
  7. Materi 11: Menjadwalkan alur kerja.
  8. Materi 12: Mengaktifkan dan menguji alur kerja.
Catatan Kredensial
Untuk membangun alur kerja ini, kita akan membutuhkan kredensial (seperti kunci API) untuk layanan seperti Airtable dan Discord.

Jika Anda mengikuti kursus resmi n8n, mereka biasanya memberikan kredensial demo melalui email saat mendaftar. Jika tidak, jangan khawatir! Kita bisa membuat akun gratis kita sendiri di Airtable dan Discord untuk mendapatkan kunci API kita sendiri.
